# Purplefinger v1.0.0

This is the first public release of Purplefinger.

I built Purplefinger as an open-source real-time face-swap system for live calls, streams, and real-time identity transformation experiments. My main goal is to keep pushing this toward a near-instant pipeline while improving streaming stability, swap quality, and overall realism.

Purplefinger has practical creative use cases, but the core thing I built it for was live face-swapped calling on social apps like WhatsApp, Telegram, Discord, and similar platforms. It also works for OBS workflows, roleplay, parody, content experiments, and live visual identity effects.

## What is in this release

- Hosted backend workflow through Render
- Local Electron client with packaged Windows portable build support
- In-app connection settings for backend URL, API token, and OBS port
- Local OBS Browser Source support on `http://localhost:7891`
- RunPod-backed GPU session orchestration
- Realtime and quality stream modes
- Portable EXE packaging flow
- Branded Windows app metadata and icon
- Simplified Purplefinger launch splash

## Project direction

The mission is not finished.

I am actively trying to make:
- streaming feel near instant
- the swap lock on faster and stay stable
- identity quality stronger
- quality mode sharper without making realtime unusable
- the entire experience easier to deploy, share, and improve
